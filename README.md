# DETOUR - Assignment 5 High-Fi React Prototype

DETOUR is a travel exploration website for college students. It helps users discover authentic, affordable local activities and then navigate a selected plan with map-based routing and contextual learning.

## Tech Stack

- React 19.2.4
- Vite 8
- JavaScript (JSX)
- React Leaflet + Leaflet
- Optional Gemini API integration (`@google/generative-ai`)

## Install and Run

```bash
npm install
npm run dev
```

Then open the local development URL shown in your terminal.

## Project Concept and Main Tasks

This prototype supports two core tasks:

1. **Task 1: Identify authentic, affordable activities**  
   Users choose preferences (vibe, budget, time, group, city) and review curated activity cards.

2. **Task 2: Plan and navigate selected activities with context**  
   Users add activities to a plan, view the route in the Explore map, and use the Learn page for neighborhood/city/country context.

## Gemini API Setup (Optional)

The Results page supports optional AI plan summaries using Gemini.  
If no key is provided, the app automatically falls back to static summaries.

Create a `.env` file at the project root:

```bash
VITE_GEMINI_API_KEY=YOUR_API_KEY_HERE
```
